# 🎮 Tic Tac Toe Android App

## 📌 Overview

This is a simple Tic Tac Toe game developed using Android Studio and Java. It is a two-player game where users can play on the same device with an interactive user interface.

---

## 🚀 Features

* 👥 Two-player gameplay
* 🎯 Automatic winner detection
* 🔄 Restart game option
* 📱 User-friendly interface

---

## 🛠️ Technologies Used

* Java
* Android Studio
* XML (for UI design)
